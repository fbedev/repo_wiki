快速開始

最常用的啟動方式

公準AI 目前主入口是 Web UI。標準做法是直接執行 gongin，系統會啟動 Web 服務並綁定到 0.0.0.0:8000。 

gongin

如果要指定 LM Studio 位址或連接埠，可直接加旗標：

gongin --lmstudio-url http://127.0.0.1:1234/v1

gongin --lmstudio-host 192.168.1.50 --lmstudio-port 1235

整合方式

如何把公準AI 整合進流程

如果你只是要啟動整個系統，最直接的方法仍然是使用 CLI 啟動 Web：

gongin --host 0.0.0.0 --port 8000

如果你要用 Python 方式啟動 Web 入口，可以直接呼叫啟動器：

from app.web_launcher import main

main([
    "--host", "0.0.0.0",
    "--port", "8000",
    "--lmstudio-url", "http://127.0.0.1:1234/v1",
])

如果你要直接使用 ASGI 伺服器：

uvicorn app.web.server:app --host 0.0.0.0 --port 8000
